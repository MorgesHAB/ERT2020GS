Detailed Description

This class manage to write and parse every needed type of Data in a RF packet
RF packet : uint8_t packet[MAX_SIZE]
Utilisation : call the methods write(Type t) and parse(Type t) for the following type :

  • float - int - char - bool - string - uint32_t - uint16_t - uint8_t Doesn't work for type - double

Compilation requirement :
These code files need to be compiled with -std=c++11 option
